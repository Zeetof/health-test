# Health Tracker - University Wellness Application

A comprehensive health tracking application designed specifically for university students to maintain a balanced lifestyle and promote better health and well-being on campus.

## 🏃‍♂️ Features

### 1. **User Profiles**
- Personal information management (age, health history, preferences)
- Secure authentication and profile customization

### 2. **Health Metrics Tracking**
- **Physical Activity**: Steps, workouts, and running logs
- **Nutrition**: Caloric intake and meal logging with detailed tracking

### 3. **Alarm and Reminder Features**
- **Activity Alarms**: Reminders for scheduled runs or workouts
- **Eating Schedule**: Notifications for meal times and eating cut-off alerts

### 4. **Goal Setting**
- Personalized goals for running, nutrition, and overall fitness
- Progress tracking and achievement milestones

### 5. **Community Features**
- Forums for sharing running tips and meal prep ideas
- Community challenges to encourage participation

### 6. **Resources and Education**
- Articles on nutrition, exercise, and healthy eating habits
- Educational content for better health awareness

### 7. **Data Privacy and Security**
- Secure storage of personal information
- Compliance with health regulations and data protection

### 8. **University Integration**
- Links to university health and wellness resources
- Campus-specific health initiatives

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- MongoDB
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd health-tracker
   ```

2. **Install dependencies**
   ```bash
   npm run install-all
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

The application will be available at:
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000

## 🛠️ Tech Stack

### Frontend
- React 18 with TypeScript
- Material-UI for modern UI components
- React Router for navigation
- Redux Toolkit for state management
- Chart.js for data visualization

### Backend
- Node.js with Express
- MongoDB with Mongoose
- JWT for authentication
- bcryptjs for password hashing
- Express rate limiting for security

### Additional Features
- Real-time notifications
- Email reminders
- Data export functionality
- Mobile-responsive design
